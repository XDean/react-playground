# Online React Editor

This is an online React editor. You can edit JSX and CSS inline and see the result immediately. See sample below.

<div className={'h-[300px] border border-solid w-8/12'}>
  <ReactEditor code={{js: CONSTANTS.HELLO_WORLD_JS}}/>
</div>

Click [HERE](./editor) to go to a pure editor.

## Init with external Resource

If you have some external resources to init the code editor, you can give `js` and `css` query in the url.
URL would like below.

<a href={'#'}>
  {(typeof window !== 'undefined' && (window.location.origin + window.location.pathname)) + "editor/?js=https://somewhere/some.js&css=https://somewhere/some.css"}
</a>

Also, you can use local file

<a href={'#'}>
  {(typeof window !== 'undefined' && (window.location.origin + window.location.pathname)) + "editor/?js=/gpfs/somewhere/some.js&css=/gpfs/somewhere/some.css"}
</a>

Form below can help you generate the page

<div className={'border border-solid mt-2 w-[500px] p-2'}>
  <OpenResource/>
</div>

  <div className={'p-2'}>
    {children}
  </div>
)
